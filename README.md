# Гайд по сборе фронта 

## Требования
- Node.js ≥ 18.x
- npm ≥ 9.x

Проверить версии:
```bash
node -v
npm -v
```

## Установка
```bash
git clone -b frontend https://github.com/kostyakaa/ITS.git
cd ITS
npm install
```

## Запуск разработки
```bash
npm run dev
```
По умолчанию: http://localhost:5173

## Сборка продакшн
```bash
npm run build
```
Сборка появится в папке `dist/`.

## Предпросмотр билда
```bash
npm run preview
```
Обычно доступен по адресу http://localhost:4173
